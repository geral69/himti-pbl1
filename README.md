# HIMTI Project Based Learning #1

## Deskripsi Proyek
Repositori ini berisi *source code* untuk **Project Based Learning 1 (PBL 1)**. 

## Struktur Halaman
Website ini dikembangkan menggunakan pendekatan berbasis komponen (*section-based*), yang terdiri dari:
* **Navbar & Hero Section** (`navbar&Hero.css`)
* **About Us** (`about.css`)
* **Events** (`events.css`)
* **Contact Us** (`contact.css`)
* **Footer** (`footer.css` - memuat tautan integrasi ke platform media sosial resmi HIMTI BINUS)

## Teknologi / Framework yang Digunakan
* **HTML5**.
* **CSS3**.

## Cara Menjalankan Secara Lokal
1. Lakukan *clone* repositori ini: `git clone https://github.com/geral69/himti-pbl1.git`
2. Buka folder `PBL1` menggunakan Visual Studio Code.
3. Jalankan file `index.html` menggunakan ekstensi **Live Server**, atau buka file tersebut langsung melalui *browser*.
